# 要件定義書

## 目的
3歳（スマホ）向けに、東京を走る電車の写真クイズを作り、GitHub Pagesで公開する。

## 対象
- 年齢: 3歳
- 端末: スマホ

## 機能要件
- クイズ形式: 4択 × 5問
- 出題範囲: 東京都に駅がある路線
  - 区分: JR / 東京メトロ / 都営 / 私鉄（第三セクター含む）
  - JRは運行系統名で統一（例: 上野東京ライン、湘南新宿ライン、中央線快速 など）
  - 除外: 新幹線 / モノレール / AGT・新交通（ゆりかもめ、日暮里・舎人ライナー など）
- 表示ラベル: 「路線名（形式）」
  - 形式が無い場合は路線名のみ
  - 現行データは全件形式あり（形式未設定はスタート無効）
- フィードバック: 画面演出のみ（音なし）
- 写真素材: ライセンスが明記された実写写真のみ（CC BY/CC BY-SA/Public domain など）
- クレジット: 出典/利用条件をアプリ内に明記
- 路線データ: 74件（追加で拡充可能）

## 非要件（MVPではやらない）
- 音声・効果音
- ランキング（オンライン）
- 文字入力が必要な回答形式

## 受け入れ条件
- スマホで「スタート → 5問 → 結果 → もう一回」が迷わず完了できる
- 各問題で写真が表示され、4択がタップで回答できる
- 選択肢ラベルが「路線名（形式）」ルールに従う
- 出題は「東京都に駅がある路線」のデータからのみ生成される
- 画像素材のクレジット/利用条件がアプリ内で確認できる
- GitHub Pagesで公開できる（静的サイト構成）
- `docs/` に要件定義書・仕様書・機能一覧が揃っている
- `README.md` にアプリ説明・起動方法・操作方法がある
- `CHANGELOG.md` がある
- `.env.sample` がある（必須無しの場合は明記）
- Docker Composeで開発実行できる

## 素材方針
- 路線一覧は Wikipedia を情報源として整理
- 写真は Wikimedia Commons など、利用条件が明記された実写写真を採用
- 各素材の出典URL・作者・ライセンスをクレジットに記載
- 画像取得の補助として `tools/fetch_commons_images.py` を利用する
